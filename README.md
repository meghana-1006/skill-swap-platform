# 🚀 Skill Swap Platform

A full-stack web application that enables users to exchange skills, connect with like-minded learners, and communicate in real-time.

---

## 🌟 Features

- 🔐 User Authentication (Register/Login with JWT)
- 👤 Profile Management (skills offered, skills wanted, pricing)
- 🤝 Match System (connect with other users)
- 💬 Real-time Chat (Socket.IO)
- 📅 Session Booking System
- 💳 Mock Payment Integration
- ⚡ Fast and responsive UI using React + Tailwind CSS

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Authentication
- JWT (JSON Web Token)
- bcrypt (password hashing)

### Real-Time Communication
- Socket.IO
